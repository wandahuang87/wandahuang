📤 Publish to GitHub

Downloads a zip of your complete portfolio with all content and photos baked in. Unzip and upload those files to GitHub — no storage limits, no browser dependency.

GitHub Pages — How to Publish

1. Go to github.com → New repository → Public

2. Upload: index.html about.html marketing.html projects.html

3. Keep admin.html on your computer only — never upload

4. Settings → Pages → Source: main / root → Save

5. Your portfolio will be live at: https://yourusername.github.io/repo-name/

Tip: For photos to show on GitHub Pages, use image URLs (e.g. upload photos to Imgur or your GitHub repo /images/ folder and paste the URL).
